# NLP-HOSPITAL-CHATBOT---MARATHI-
NLP Knowledge Artifact Repository for Hospital Services Chatbot (Marathi) - CCE Component I
# Project Overview

---

This Knowledge Artifact Repository presents the Natural Language Processing (NLP) concepts used in the development of a **Hospital Services Chatbot – An AI-Based Administrative Assistance System (Marathi Language)**. The chatbot helps patients obtain information regarding appointments, departments, OPD timings, billing, insurance, doctor availability, and emergency services by understanding Marathi-language queries using NLP techniques.

---

# 🎯 Objectives

---

- Develop a Marathi language hospital chatbot.
- Apply NLP concepts to understand patient queries.
- Reduce hospital reception workload.
- Provide quick responses to administrative questions.
- Demonstrate NLP concepts through a real-world application.
